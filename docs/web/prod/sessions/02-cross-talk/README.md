# 異分野クロストーク

様々な技術が複雑に絡み合う現在では、フロントエンド・バックエンド両方の知識が必要とされることがあります。このセッションでは、バックエンドのエキスパートがフロントエンド・エンジニアへ、逆にフロントエンドのエキスパートがバックエンドエンジニアへ、知っておくと便利な異分野の知識を伝授します。

フロントエンドからバックエンドまで、様々な技術トピックを扱う Google Developer Groups (GDG) ならではセッションです。GDGならではの視点から、新しい学びを楽しみましょう！
