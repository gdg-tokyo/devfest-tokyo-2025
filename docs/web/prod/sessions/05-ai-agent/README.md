# AI エージェント 入門

生成AI が急速に普及し、私たちの生活を変えようとしています。この変化の中核となる技術が「AI エージェント」です。この入門セッションでは、AI エージェントが生活にもたらす変化とその価値について議論し、実際に AI エージェント 開発を始めるために必要な、仕組みや開発プロセスについて、わかりやすく解説していきます。

AI初心者の皆さんも、このセッションに参加することで、AI エージェント開発の開発の準備を整えましょう！
