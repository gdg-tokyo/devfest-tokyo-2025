# [懇親会LT] MediaPipeを用いたイラストのトレーニングの自動化

MediaPipeが苦手な2次元イラストのセグメンテーションに対し、既存データをMediaPipeで前処理して学習用マスクを自動生成し、モデルをトレーニングする手法を提案します。

## Speaker

### 鈴木崇博 さん / 東京国際工科専門職大学工科学部情報工学科 学生

過去には、検索エンジンを搭載した Stable Diffusion の拡張ライブラリである auto_diffusers を開発しました。こうした個人開発の経験を基盤に、AI と宇宙工学の両分野を結び付け、社会や技術に貢献できる研究をしたいと考えています。
